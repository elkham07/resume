About MovieHub

MovieHub is a learning project inspired by Netflix.

was created as an initial step toward a full-fledged platform where users will be able

not only to search for but also to watch movies online.



In the future, it is planned to add features such as user registration, favorites, and a recommendation system.

The goal of the project is to demonstrate the fundamentals of web development using HTML and CSS, along with basic APIs,

and to gradually expand it with JavaScript.
